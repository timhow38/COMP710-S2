# Start Instructions:

Install Requirements
> npm i angular/cli -g

Create Framework
> ng new angular-tour-of-heroes
> Routing = No
> Select = SCSS

Switch to DIR
> cd angular-tour-of-heroes

Serve Web Server
> ng serve --open

HeroApp will now be running [localhost:4200]
